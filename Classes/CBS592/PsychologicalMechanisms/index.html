Module overview

The goal of this module is for you to become familiar with some of the core cognitive components that are relevant in language processing, and of the challenges in how we can measure and identify them. By the end of the module, hopefully you will be able to do the following things:

  • Identify core cognitive abilities that are relevant in psycholinguistics;
  • Explain how these abilities can be measured;
  • Explain how these cognitive abilities can impact language processing.

Module instructions

This module includes five tasks. For the first task of this module, you have to perform some tests on some volunteers and then do some reflection about your findings. After you complete that, the rest of the tasks can be completed in any order.

Module activities

  1. Test your friends (3 hours)
  2. Several basic psychological mechanisms
    • Working memory (2 hours)
    • Executive function (2 hours)
    • Top-down vs. bottom-up processing (1 hour)
  3. How to score cognitive tasks (3 hours)
  4. Optional: discussion topics
