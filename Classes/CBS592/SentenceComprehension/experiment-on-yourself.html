Experiment on yourself (1 hour)

↵ Back to module homepage

First, let's do an experiment.

There are two ways for you to access this experiment: you can do it online via a website, or you can download the experiment and run it on your computer using DMDX. Do the experiment either way, and then continue to the questions below.

Option 1: Do the experiment online

To do the experiment online, go to the web experiment and follow the instructions there. Note: this web platform is sometimes inaccessible in China. If you can't open the page, you can consider using a VPN, or using the DMDX version of the experiment below.

Option 2: Download the experiment and run it on DMDX

There are two versions of the experiment (these are two Latin Square lists; to learn more about Latin Square, complete the "Building a priming experiment" module):

  • spr_garden_List2.rtf
  • spr_garden.rtf

Download either of these versions, and run the experiment using DMDX. (To learn how to install and use DMDX, complete the first section of the "Priming" module.)

What do you think this experiment was about? What do you think I wanted to test, and what results do I want to compare? What do you predict the results might show?

When you have finished these activities, continue to the next section of the module: "Reflection on the experiment".
