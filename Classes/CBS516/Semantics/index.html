Module overview

The goal of this module is to learn some basic concepts of compositional semantics and use them to examine semantic differences between languages. By the end of the module, hopefully you will be able to do the following things:

  • Explain the formal semantics perspective on what "meaning" is;
  • Identify differences between semantic phenomena in Chinese and English.

Module instructions

This module includes two tasks. For each task, you will need to read something and then answer some questions; some of these questions may require a long time to think about. These tasks are meant to be done in order (i.e., the intention is for you to not start one task until you have done the previous task; when this was taught via an LMS the later modules would be "locked" until the student completed the previous ones). To receive credit for completing this module, you must complete all the tasks at a satisfactory level of quality.

Next to each task I have written an estimate of how much time you might need to complete the task. This is, of course, a rough estimate, and the real time may be different for different students.

Module activities

  1. What is meaning? (5 hours)
  2. Some semantic phenomena (3 hours)

Suggested discussion topics/activities

  • Come up with examples of sentences that formally have a distributive-collective ambiguity but that are unambiguous because of world knowledge (e.g. "A bomb exploded in every city"—technically this could mean the same bomb [i.e. there is a bomb and that bomb exploded in every city], but that doesn't make sense because we know that once a bomb explodes in one city it's destroyed and thus the can't blow up in another city]). Get one distributive and one collective example.
  • Come up with real-world examples of times when a semantic ambiguity could lead to a disagreement or misunderstanding.
