Module overview

The goal of this module is for you to know how to make informative graphs of data. By the end of the module, hopefully you will be able to do the following things:

  • Explain what information is useful in a graph, and why graphs that only show summary statistics are insufficient for certain purposes;
  • Use informative graphs to illustrate different kinds of data patterns;
  • (advanced) Make an informative graph of a dataset.

Module instructions

This module includes one task, plus one task for advanced completion.

Module activities

  1. The importance of informative visualizations (6 hours)
  2. Advanced level (3 hours)
