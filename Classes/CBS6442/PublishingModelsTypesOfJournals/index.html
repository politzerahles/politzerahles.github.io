Publishing models and types of journals (4.5-6.5 hours)

↵ Back to class homepage

Module overview

The goal of this module is for you to understand different publishing models and different types of journals, which will help you in the future when you have to select which journals to choose to submit papers to. By the end of the module, hopefully you will be able to do the following things:

  • Explain what open-access publishing is, and different kinds of open access;
  • Explain what registered reports are;
  • Identify predatory publishers;
  • (advanced) Select a journal to submit a hypothetical paper to.

Module instructions

This module includes three tasks (plus one task for advanced completion).

Module activities

  1. Open access (2 hours)
  2. Registered reports (1 hour)
  3. Predatory publishers (1.5 hours)
  4. Advanced level (4 hours)
