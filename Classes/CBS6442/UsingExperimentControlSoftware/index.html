Using experiment control software (5-16 hours)

↵ Back to class homepage

Module overview

The goal of this module is for you to learn how to set up basic experiments using experimental control software. This module focuses on DMDX just because it is free and relatively easy (for certain kinds of experiments), but interested students or teachers could swap out DMDX for other popular software options like PsychoPy, PCIbex, Open Sesame, E-Prime, etc. By the end of the module, hopefully you will be able to do the following things:

  • Find and compare different experimental control programs and judge which ones are appropriate for which situations;
  • Program a simple experiment using DMDX;
  • (advanced) Replicate the experiment design of a published study in an experiment control program.

Module instructions

This module includes two tasks that you need to do, as well as two tutorials that are not necessary but that may help you complete the tasks.

Module activities

  1. What is experiment control software? (2 hours)
  2. Tutorials
    1. Priming in DMDX (2 hours)
    2. Self-paced reading in DMDX (1 hour)
  3. Make a simple DMDX experiment (1 hour)
  4. Advanced level (5-10 hours)
