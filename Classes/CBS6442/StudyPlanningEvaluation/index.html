Study planning and evaluation: power, effect size, and confidence intervals (9-11 hours)

↵ Back to class homepage

Module overview

The goal of this module is for you to understand how to use effect sizes, confidence intervals, and power analysis to plan studies and to evaluate results. By the end of the module, hopefully you will be able to do the following things:

  • Explain what effect size is, and some common effect size measures;
  • Explain what power analysis is for and how to do it;
  • Interpret confidence intervals correctly, and know how to use them to present results;
  • (advanced) Plan a study design using principles of effect size and power analysis.

Module instructions

This module includes three tasks, plus one task for advanced completion.

The first task ("Effect sizes") must be done first, because the other tasks depend on the concepts introduced there. The remaining tasks can be done in any order.

Module activities

  1. Effect sizes (4 hours)
    • Confidence intervals (2 hours)
    • Power (3 hours)
  3. Advanced level (2 hours)
