ANOVA and factorial designs (8-12 hours)

Module overview

The goal of this module is for you to understand what situations ANOVA is useful for, to understand different types of effects in a factorial research design, and to know when and how to use multiple comparison correction. By the end of the module, hopefully you will be able to do the following things:

  • Explain what ANOVA shows and know when to use it;
  • Recognize and make predictions about the different effects in a factorial design;
  • Recognize when multiple comparison correction is needed;
  • (advanced) Carry out an ANOVA on real data and make conclusions from the results.

Module instructions

This module includes three tasks, plus one task for advanced completion.

Module activities

  1. What ANOVA is (2 hours)
  2. Factorial designs (4 hours)
  3. Multiple comparisons (2 hours)
  4. Advanced level (4 hours)
